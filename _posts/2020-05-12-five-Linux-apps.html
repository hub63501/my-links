Here are five linux apps to try if your a linux user.

  1. htop: a good tool for analyzing network traffic.

  2. fondo: Wallpapers for everyone. A program that looks great and makes your desktop look great.

  3. Odio: A really nice looking application that streams radio stations from around the world. If you've ever tried the mobile app Tunein then this is like the desktop version of that but in my view a whole lot better.

  4. Vivaldi: a very underated web browser that keeps on getting better with each update. Also the mobile web browser is'nt bad either and is one browser that still ocupies the top tab function.

  5. Laverna: a clean looking note taking app with encryption enabled.
